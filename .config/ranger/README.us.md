## :wrench: Get Started 

> For more options [click here](https://github.com/ranger/ranger#installing)

- Python

```bash
pip3 install ranger-fm
```

- Yay

```bash
yay -S ranger
```

copy the files to `~/.config/ranger`

```bash
cp rc.conf install.sh ~/.config/ranger
```
